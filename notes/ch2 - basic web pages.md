# Chapter 2 notes

- MDN (the Mozilla Developer Network) is an excellent resource for html references

A good template looks similar to the following, without any CSS

```html
<!DOCTYPE html>
<html lang='en'>
  <head>
    <meta charset='UTF-8'/>
    <title>Some Web Page</title>
  </head>
  <body>
    <h1>Some Web Page</h1>
    <!-- Rest of the page content -->
  </body>
</html>
```
